# YouDrive 

Sitio web de una empresa ficticia de alquiler de vehículos, hecho como proyecto para la materia Programación 3.

## ¿Qué es?

YouDrive es una página que simula la presencia web de una empresa de alquiler de autos. Tiene su propio logo, secciones con los servicios y vehículos disponibles, y un formulario para que el cliente pueda hacer una reserva.

## Páginas

- **index.html** — Inicio: imagen de bienvenida, servicios, vehículos destacados, sobre nosotros y footer con contacto.
- **reserva.html** — Formulario de reserva: datos personales, tipo de vehículo, fechas, sucursal, método de pago y extras opcionales.

## Tecnologías

- HTML5
- CSS3 (sin frameworks)

## Estructura

```
 proyecto
├── index.html
├── reserva.html
├──  styles
│   ├── global.css
│   └── reserva.css
└──  img
```

## Responsive

El sitio se adapta a distintos tamaños de pantalla: computadora, tablet, celular y pantallas muy pequeñas.
